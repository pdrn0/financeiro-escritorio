# Financeiro - Escritorio

Controle de entradas e saidas com relatorio de divisao entre socios.

## Deploy no Railway
1. Acesse railway.app e conecte este repositorio
2. Adicione um Volume com mount path `/data`
3. Acesse a URL gerada

## Rodar localmente
```
npm install
npm start
```
Acesse: http://localhost:3000
